A retro 70s-inspired personal portfolio theme with modern WordPress APIs.

== Description ==

Earthbound is a modern WordPress block theme that brings the warm, earthy aesthetics of the 1970s to your personal portfolio. Built with the latest WordPress APIs including Full Site Editing and the Interactivity API.

Features:

* Full Site Editing support with customizable templates and template parts
* Four beautiful style variations: Harvest Gold, Desert Sunset, Forest Grove, and Ocean Tide
* Custom navigation block with Interactivity API for smooth, accessible interactions
* Projects custom post type for showcasing your work
* GitHub and WordPress Trac feed integration
* Fully accessible with keyboard navigation and screen reader support
* Reduced motion support for users who prefer less animation
* Fluid typography that scales smoothly across all screen sizes
* Self-hosted fonts for optimal performance

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support Full Site Editing? =

Yes! Earthbound is a block theme that fully supports the Site Editor.

= How do I add my GitHub contributions? =

Navigate to Appearance > Customize > Earthbound Settings and enter your GitHub username.

= How do I change the color scheme? =

In the Site Editor, click on Styles in the top right, then browse the available style variations.
